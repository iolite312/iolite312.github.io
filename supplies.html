Wat heb je nodig

Zoals ik eerder heb gezegd heb je niet heel veel nodig om te beginnen. Zo hebt je verschillende progamma's en manieren om code te kunnen schrijven. Maar ook bijvoorbeeld verschillende progamma's die gespecialiseerd zijn in bepaalde dingen te ontwikkelen zoals games, of websites.

Code schrijven

Er zijn verschillende manieren hoe je code kan schrijven. Je kan bijvoorbeeld een progamma gebruiken die speciaal is gemaakt voor coderen zoals Visual Studio Code. Je kan ook in je browser gaan naar een website waar je kan coderen en dat is vooral handig als je een nieuwe codeer taal wilt uit testen of als je wilt beginnen met coderen maar nog niks wilt installeren. Een paar websites waar je dat kan doen zijn: Stackblitz en Replit. Hiernaast heb ik nog wat opties als je een progamma wilt installeren inplaats van in je browser werken.

  • Visual studio
  • Visual Studio Code
  • Notepad++
  • Sublime text

  • Of een ander tekstbewerkingsprogamma

Hoe bekijk je je werk?

We zijn er nog niet wat je ook nodig hebt is een plek om je website te bekijken als hem gaat maken. Er zijn verschillende manieren hoe je dat kan doen. Je kan met behulp van extensies in Visual studio code (VS Code) kan je een webpagina in je browser laten zien en die automatisch ververst als je een aanpassing heb gedaan aan je code. Wat je ook kan doen is een progamma zoals XAMPP installeren om je pagina te laten zien in je browser alleen daarvan is een nadeel dat je na elke aanpassing je pagina handmatig moet verversen. En als je het niet lukt om 1 van de bovenstaande dingen te gebruiken is er altijd nog een laatste oplossing en dat is het bestand dubbelklikken in Verkenner dat hij hem dan opent in de browser, alleen hetzelfde nadeel geld net zoals bij XAMPP je pagina zal niet automatisch verversen.

Standaarden die er zijn

Natuurlijk zijn er ook standaarden tijdens het coderen die spreek je meestal af met je klasgenoten/collega's. Een voorbeeld van een code standaard is camel case of pascal case het verschil tussen deze twee zijn als je een variable een naam geef met camel case bijvoorbeeld dan kan het zijn bananenDoos maar met pascal case is het BananenDoos. Maar er zijn ook standaarden per codeertaal als je daar meer over wilt weten is google je grootste vriend.
